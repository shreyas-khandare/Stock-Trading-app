# Stock Trading App

A full-stack stock trading simulation platform inspired by modern trading applications like Zerodha. The application allows users to explore stocks, manage holdings, track positions, and simulate stock trading activities through an interactive dashboard.

---

## Tech Stack

### Frontend
- React.js
- CSS
- JavaScript

### Dashboard
- React.js
- Context API
- Chart Libraries

### Backend
- Node.js
- Express.js
- MongoDB

---

## Project Structure

```bash
Stock-trading-app/
│
├── backend/      # Backend APIs and server
├── dashboard/    # Trading dashboard UI
├── frontend/     # Landing website/frontend
```

---

## Features

### Trading Dashboard
- Watchlist management
- Holdings tracking
- Positions overview
- Orders management
- Buy/Sell stock simulation
- Portfolio charts and analytics

### Frontend Website
- Responsive landing pages
- Navigation and UI components
- Product/service showcase

### Backend
- REST API integration
- Transaction handling
- Portfolio data management
- Server-side business logic

## Installation

### Clone the Repository

```bash
git clone https://github.com/shreyas-khandare/stock-trading-app.git
cd stock-trading-app
```

---

## Backend Setup

```bash
cd backend
npm install
npm start
```

---

## Frontend Setup

```bash
cd frontend
npm install
npm start
```

---

## Dashboard Setup

```bash
cd dashboard
npm install
npm start
```

---

## Environment Variables

Create a `.env` file inside the `backend` folder and add your environment variables.

Example:

```env
PORT=5000
MONGO_URI=your_mongodb_connection_string
```
